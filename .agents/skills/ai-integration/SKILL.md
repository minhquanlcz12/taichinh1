# AI Integration & Automation Skill

This skill provides patterns for making web applications "intelligent" using Large Language Models.

## 1. Prompt Engineering for Web
- **JSON Output Mode**: Always instruct the AI to return data in a structured JSON format to avoid parsing errors.
- **Context Injection**: Patterns for sending the current application state (e.g., transaction history, user prompts) to the AI for analysis.

## 2. API Management
- **Proxy Handling**: Instructions for securing API keys and handling server-side requests.
- **Rate-Limit Mitigation**: Implementation of retry logic and batching for high-frequency requests.

## 3. Automation Flows
- **Content Generation**: Automated workflows for writing SEO articles, descriptions, or generating images based on app data.
- **Data Analysis**: AI-powered insights into financial records or user logs.
