# SEO & Performance Optimization Skill

This skill ensures your web application is highly discoverable and performant.

## 1. Professional SEO
- **Dynamic Meta Tags**: Logic for updating `<title>`, `<meta name="description">`, and OpenGraph tags dynamically.
- **Semantic Structure**: Utilizing H1-H6 hierarchy and ARIA labels for accessibility and search ranking.
- **Sitemap & robots.txt**: Patterns for guiding search engine crawlers.

## 2. Performance Engineering
- **Asset Optimization**: Compressing images to WebP and minifying CSS/JS.
- **Lazy Loading**: Implementing `loading="lazy"` for all non-critical images and assets.
- **Cache Control**: Advanced versioning and caching strategies for static assets.
